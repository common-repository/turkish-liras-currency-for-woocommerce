== Installation ==

= Türkçe =
1. WordPress Eklenti sayfasından eklentiyi indirerek veya WordPress Admin sayfasından Eklentiler > Yeni Ekle sayfasına girerek arama yerine WooCommerce için Türk Lirası Simgesi yazarak eklentiyi kurabilirsiniz
2. Eklentiyi aktif ediyorsunuz.

= English =

1. Install Turkish Liras Currency for WooCommerce either via the WordPress.org plugin directory, or by uploading the files to your server.
2. After activating WooCommerce için Türk Lirası Simgesi on plugins page.
3. Say Hi to New Turkish Liras Currency!

== Frequently Asked Questions ==

= Sitemde Türk Lirasını Nasıl Gösterebilirim ? =
- Eklentiyi aktif ettinten sonra http://siteniz.com/wp-admin/admin.php?page=wc-settings sayfasına giriyorsunuz
- "Para Birimleri" kısmındaki açılır menüden "TL Birimi (t)"i seçiyorsunuz
- "Ayarlarımı Kaydet" diyorsunuz.

= How can I active the Turkish Liras Currency ? =
- After activing the plugin, go to http://yoursite.com/wp-admin/admin.php?page=wc-settings
- Find the "Price Currency", select "TL Birimi (t)"
- Click to "Save" button
